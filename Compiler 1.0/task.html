Постановка задачи

Функции языка Rust представляют собой именованные конструкции, включающие параметры, возвращаемое значение и логическое тело. Такие конструкции используются для повторного использования кода, структурирования логики и соблюдения принципов модульности.

Формат записи функции в Rust для моего задания включает:

  • Ключевое слово fn, обозначающее начало функции;
  • Имя функции (идентификатор);
  • Список параметров с их типами в круглых скобках;
  • Возвращаемый тип после стрелки ->;
  • Тело функции в фигурных скобках с обязательным оператором return.

Примеры корректных определений функций:

  1. Однострочная функция без параметров:
    fn main() -> i32 { return 0; };
  2. Функция с параметрами:
    fn sum(a: i32, b: i32) -> i32 { return a + b; };
  3. Функция с выражением:
    fn calc(a: i32, b: i32, c: i32) -> i32 { return a + b * (c - a); };

Лексический и синтаксический анализатор на языке C# обрабатывает следующие компоненты функции:

  • fn, return — ключевые слова;
  • Идентификаторы — имена функций и переменных;
  • Типы переменных: i8, i16, i32, i64, i128, f32, f64, bool, char, &str;
  • Символы: (, ), {, }, ,, :, ;, ->;
  • Арифметические операторы: +, -, *, /;
  • Целочисленные значения.
