Справка

Файл

  • Создать — очищает текущее окно редактирования для создания нового файла.
  • Открыть — открывает один или несколько текстовых файлов для анализа.
  • Сохранить — сохраняет текущий файл, если он уже имеет путь.
  • Сохранить как — сохраняет текущий файл с выбором имени и расположения.
  • Выход — завершает работу программы, с предупреждением о несохранённых данных.

Правка

  • Отменить — отменяет последнее действие в редакторе.
  • Повторить — возвращает отменённое действие.
  • Вырезать — вырезает выделенный текст.
  • Копировать — копирует выделенный текст в буфер.
  • Вставить — вставляет текст из буфера обмена.
  • Удалить — удаляет выделенный текст.
  • Выделить все — выделяет весь текст в текущем редакторе.

Текст

  • Постановка задачи — описание цели программы и функциональности.
  • Грамматика — отображает используемую грамматику в нотации Хомского.
  • Классификация грамматики — показывает тип грамматики по классификации Хомского.
  • Метод анализа — описание метода лексического и синтаксического анализа.
  • Диагностика и нейтрализация ошибок — пояснение обработки ошибок в программе.
  • Тестовый пример — пример корректной функции для анализа.
  • Список литературы — перечень использованных источников.
  • Исходный код программы — текст реализованной программы на C#.

Пуск

  • Пуск — запускает анализ текущего файла: сначала лексический, затем синтаксический.

Справка

  • Вызов справки — открывает текущую HTML-страницу справки.
  • О программе — отображает информацию об авторе и версии программы.
